# Running_Fatigue_Analysis

# Multivariate Analysis of Running Fatigue and Footwear Performance

## Overview
This project investigates how fatigue develops during real-world running using physiological, biomechanical, and environmental data. The goal is to identify patterns in fatigue and explore how these findings can inform footwear design and performance optimization.

## Motivation
As both a runner and biomedical engineering student, I am interested in understanding how performance changes over the course of a run and how footwear interacts with fatigue. This project aims to bridge the gap between data-driven performance analysis and product innovation in running shoes.

## Data Collected
For each run, the following data is recorded:

- Heart Rate (HR)
- Pace
- Distance
- Cadence
- Rate of Perceived Exertion (RPE)
- Shoe Used
- Temperature
- UV Index
- Notes (qualitative observations)

## Objectives
- Identify key indicators of fatigue (HR drift, pace changes, RPE)
- Develop a fatigue score combining multiple variables
- Analyze the impact of environmental factors (temperature, UV)
- Compare performance across different running shoes
- Translate findings into insights for footwear design

## Methodology
1. Data collection from real-world runs
2. Data cleaning and preprocessing using Python (pandas)
3. Visualization of trends (HR, pace, fatigue)
4. Development of a fatigue scoring model
5. Comparative analysis across shoes and conditions

## Key Questions
- How does fatigue appear in physiological and performance data?
- What variables change first as fatigue increases?
- How do environmental factors affect fatigue?
- Do different shoes influence fatigue patterns?
